# CleanFile Studio

This is a simple landing page + intake form for your AI data cleanup service.

## Customize in 3 minutes

1. Update your email address in `intake.html` (search for `youremail@example.com`).
2. Edit the business name or tagline in `index.html` if you want a different brand.
3. Open `index.html` in a browser to preview.

## Files

- `index.html`: Landing page
- `intake.html`: Client intake form
- `styles.css`: Styles
- `script.js`: Reveal animations + current year

## Optional upgrades later

- Replace the mailto form with a Google Form or Formspree.
- Add a portfolio section with real client samples.
- Connect to the OpenAI API for automation.
